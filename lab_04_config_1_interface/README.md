修改内容:

2. my_test.sv 里用 `uvm_config_db#(int)::set` 设置这个配置项的值 -> 在 `pre_randomize()` 里设置, 以确保能在随机化之前设置

2. *根据其他类成员的值来设置随机化变量的约束*
3. *执行一些预处理逻辑*
4. *调用其他函数或方法来准备随机化过程*
